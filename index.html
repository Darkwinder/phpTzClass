PhpTZclass

PHP-Klasse zur dynamischen Generierung von Taktischen Zeichen für den Katastrophenschutz

View on GitHub Download .zip Download .tar.gz

Anwendung:

$img = new TaktischeZeichen("Taktische Einheit","Technische Hilfeleistung","blau",200);

Der erste Parameter gibt das Grundzeichen an: [[Taktische Formation|Taktische Einheit|Taktischer Verband|Dienststelle]|[Befehlsstelle]|[Stelle|Einrichtung]|[Person]|[Maßnahme]|[Anlass|Ereignis]|[Gefahr]|[ortsgebunden|ortsfest]|[Gebäude]]

Der zweite Parameter gibt das Symbol innerhalb des Grundzeichens an - die Fachaufgabe, z.B. "Sprengen", "Technische Hilfeleistung" und alles andere, das man zu tz.xml -> DV1102 -> fachaufgaben hinzugefügt hat.

Der dritte Parameter ist die Grundfarbe des Zeichens: [rot|blau|weiß|gelb|grün|orange]

Der vierte Parameter ist die gewünschte Zeichenbreite in Pixel.

Mit

$img->output('svg');

wird das Zeichen ausgegeben, wobei der Funktionswert "svg", "jpg", "jpeg", "png" oder "gif" sein kann, wird nichts angegeben gilt "svg"

Beispiele:

Polizist: $img = new TaktischeZeichen("Person","","grün",200); $img->output('svg');

Einheit der Feuerwehr: $img = new TaktischeZeichen("Taktische Einheit","","rot",100); $img->output('svg');

Bombenentschärfungseinheit: $img = new TaktischeZeichen("Taktische Einheit","Kampfmittelräumung","grün",150); $img->output('svg');

Usage:

$img = new TaktischeZeichen("Taktische Einheit","Technische Hilfeleistung","blau",200);

First Value can be: [[Taktische Formation|Taktische Einheit|Taktischer Verband|Dienststelle]|[Befehlsstelle]|[Stelle|Einrichtung]|[Person]|[Maßnahme]|[Anlass|Ereignis]|[Gefahr]|[ortsgebunden|ortsfest]|[Gebäude]] Second Value can be every symbol which can be shown within the shape, e.g. "Sprengen", "Technische Hilfeleistung" and everything added to tz.xml -> DV1102 -> fachaufgaben Third Value can be: [rot|blau|weiß|gelb|grün|orange] Fourth Value is the desired width in pixel.

$img->output('svg'); // can be "svg", "jpg", "jpeg", "png", "gif", default is "svg"

Examples:

Policeofficer: $img = new TaktischeZeichen("Person","","grün",200); $img->output('svg');

Fire Rescue Unit: $img = new TaktischeZeichen("Taktische Einheit","","rot",100); $img->output('svg');

Bomb Squad: $img = new TaktischeZeichen("Taktische Einheit","Kampfmittelräumung","grün",150); $img->output('svg');
